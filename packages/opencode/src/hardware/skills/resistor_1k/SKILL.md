# Skill: 1k Ohm Resistor (1000Ω)

## Deep Intelligence: Engineering Wisdom
- **The "Gold Standard" of Pull-ups**: 1kΩ to 10kΩ are the most common values for pull-up resistors on digital input pins. 1kΩ provides a "stronger" pull-up (less susceptible to noise) than 10kΩ.
- **Current Limiting**: At 5V, a 1kΩ resistor limits current to 5mA (I=V/R). This is safe for most logic signals but might be too dim for a standard LED (where 220Ω to 330Ω is preferred).
- **Ohm's Law Mastery**: Always calculate power dissipation (P = V² / R). For a 0.25W resistor at 12V, P = 144 / 1000 = 0.144W (Safe).
- **Tolerance**: The gold band signifies ±5% tolerance, meaning the actual resistance can be between 950Ω and 1050Ω.

## Hardware Specs
- **Value**: 1kΩ (1000 Ohms).
- **Color Bands**: Brown, Black, Red, Gold.
- **Power Rating**: Usually 1/4 Watt (0.25W) for through-hole.
- **Package**: Axial (Through-Hole).

## Pro-Grade Electronics
- **Lead Forming**: Use a lead-bending tool or pliers to make clean 90-degree bends for a professional look on breadboards/PCBs.
- **Direction**: Resistors are non-polarized (no positive/negative), but it's good practice to align all resistors in the same direction so they can be read easily.
